~~~~~~~~~~~~ READ ME ~~~~~~~~~~~~
by Bryan Tragle
Jul 8th, 2011

This archive contains 2 folders. Obviously, the FR folder stands for the French version of the skin, and the ENG folder stands for the English one. The only differences in those two versions are the language in the installation instructions, and the skin's banner (the "hostname.bmp" file).

In order to know how to install this skin, just read the "installation.txt" file in either the ENG folder or the FR folder.

-----

Cette archive ZIP contient 2 dossiers. Comme leur nom l'indique, le dossier FR contient la version française du thème, et le dossier ENG la version anglaise. Les seules différences entre ces deux dossiers sont la langue dans laquelle est écrite le fichier "installation.txt", et la bannière du thème (l'image "hostname.bmp").

Vous trouverez les instructions d'installation du thème dans le fichier "installation.txt" situé dans chacun des deux dossiers ; bien entendu, si vous souhaitez avoir une version anglaise, lisez celui situé dans le dossier ENG, et lisez celui situé dans le dossier FR pour avoir les instructions en Français !
